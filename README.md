## Install

```bash
# Using npm
npm i @belvoly-vue-aioa/bvant -S

# Using yarn
yarn add @belvoly-vue-aioa/bvant
```

## Quickstart

```js
import Vue from 'vue';
import { Button } from '@belvoly-vue-aioa/bvant';
import '@belvoly-vue-aioa/bvant/lib/index.css';

Vue.use(Button);
```
