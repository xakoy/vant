<h3 align="center" style="margin: 30px 0 35px;">基于vant修改的使用于移动端UI</h3>

---

## 安装

```bash
# 通过 npm 安装
npm i vant -S

# 通过 yarn 安装
yarn add vant
```

## 快速上手

```js
import Vue from 'vue';
import { Button } from '@belvoly-vue-aioa/bvant';
import '@belvoly-vue-aioa/bvant/lib/index.css';

Vue.use(Button);
```
